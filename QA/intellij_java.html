First Java Program


Introduction

Hello and welcome to this beginner tutorial on Java programming. In this session, we'll use IntelliJ IDEA as our Integrated Development Environment (IDE) to write and run our first Java program. We'll cover the basics, including downloading IntelliJ, setting up a workspace, and creating a Java project to generate our first program.


What is an IDE?

Before we dive into IntelliJ, let's understand what an IDE is. An IDE, or Integrated Development Environment, is a software application that provides comprehensive tools for software development. It serves as a centralized platform where developers can write, compile, and run their code efficiently. While you can use simple editors like Notepad, IDEs offer advanced features.


1. Downloading IntelliJ IDEA

Enter the IntelliJ website and choose the version suitable for your operating system. After downloading the installation file, complete the setup process.


2. Open Workbench

Once IntelliJ starts, it will present you with the Workbench. The Workbench is the main development area where you'll manage and organize your projects. Familiarize yourself with the different sections, such as the project explorer, editor, and console.


Creating Your First Java Program

Now that IntelliJ is set up, let's move on to creating and running a simple Java program.


3. Create a Java Project

- Click on "New Project" within the IDE.

- Provide a project name (e.g., Learn Java).

- Choose the location where you want to store your project.

- Choose Java as the programming language.

- Click on the JDK dropdown and choose "Download JDK".


4. Write and Run Your Program

- Write a simple Java program under the main method - public static void main(String[] args) :

System.out.println("Hello World!");

- Run the program by clicking the green "Run" button.

- Save the project by typing ctrl + s.


Here is the complete code: 

class Main {
          public static void main(String[] args) {
              System.out.println("Hello World!"); 
          }
      }

Congratulations! You've successfully created and run your first Java program using IntelliJ. Feel free to explore more features of IntelliJ as you continue your Java programming journey. Whether you're on Windows or Mac, the steps remain the same in IntelliJ, making it a platform-independent development environment.
